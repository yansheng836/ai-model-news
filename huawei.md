# 华为

## 公司简介

| 项目 | 内容 |
|------|------|
| 国家 | 中国 |
| 总部 | 深圳 |
| 成立时间 | 1987年 |
| 创始人 | 任正非 |
| 定位 | 全球领先的ICT基础设施和智能终端提供商，依托昇腾AI芯片生态推进大模型 |

## 大模型产品

### 盘古 / Pangu 系列

| 版本 | 发布时间 | 主要特点 |
|------|----------|----------|
| 盘古大模型 1.0 | 2021年 | 首次发布，包含NLP和CV模型 |
| 盘古大模型持续迭代 | 2022年 | 聚焦行业应用（矿山、药物分子、气象等） |
| 盘古大模型 3.0 | 2023年7月 | 华为云开发者大会发布，定位"行业大模型" |

### 盘古大模型 3.0 三层架构

| 层级 | 说明 |
|------|------|
| L0 基础模型 | NLP、CV、多模态、预测、科学计算五大能力 |
| L1 行业模型 | 面向特定行业的模型 |
| L2 场景模型 | 面向具体业务场景的模型 |

### 技术特点

- 依托**昇腾（Ascend）AI芯片**和**华为云**基础设施
- 聚焦**行业大模型**落地（政务、金融、矿山、气象、药物研发等）
- 不以通用对话为主要竞争方向，强调行业赋能
