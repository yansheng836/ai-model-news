# 智谱AI

## 公司简介

| 项目 | 内容 |
|------|------|
| 国家 | 中国 |
| 总部 | 北京 |
| 成立时间 | 2019年 |
| 创始人 | 唐杰（清华大学教授） |
| 定位 | 源自清华大学的AI公司，专注大模型研发，推出GLM系列 |

## 大模型产品

### ChatGLM / GLM 系列

| 版本 | 发布时间 | 主要特点 |
|------|----------|----------|
| ChatGLM-6B | 2023年3月 | 首个开源中英双语对话模型，62亿参数，基于GLM架构 |
| ChatGLM2-6B | 2023年6月 | 第二代开源模型，性能大幅提升，支持32K上下文 |
| ChatGLM3 | 2023年10月 | 第三代模型，支持工具调用、代码执行、Agent功能 |
| GLM-4 系列 | 2024年1月 | 包括GLM-4、GLM-4V（多模态）、GLM-4-AllTools等 |
| GLM-4-Plus / GLM-4V-Plus | 2024年 | 持续迭代升级版本 |
| GLM-Z1 | 2025年1月 | 新的推理模型 |
| GLM-4.5 系列 | 2025年 | 包括GLM-4.5、GLM-4.5-Air等 |

### 核心亮点

- **GLM架构**：自研通用语言模型架构（General Language Model）
- **开源生态**：ChatGLM系列在开源社区有广泛影响力
- **学术背景**：团队源自清华大学，学术研究与工程落地并重
- **多模态**：GLM-4V支持图文理解
- **Agent能力**：GLM-4-AllTools支持工具调用和自主任务执行
